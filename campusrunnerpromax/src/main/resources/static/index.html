使用说明
  1. 注册/登录：点击右上角注册新账号或登录已有账号
  2. 发布任务：登录后点击"发布任务"，填写任务信息并上传图片
  3. 搜索任务：使用搜索栏按关键词、分类、状态筛选任务
  4. 接取任务：浏览任务列表，点击"接取任务"按钮
  5. 完成任务：接取任务后可以点击"完成任务"标记为完成
  6. 我的任务：查看我发布的任务和接取的任务
  7. 个人中心：修改个人信息、更换头像
常见问题

登录后，点击导航栏右上角的"发布任务"按钮，填写任务标题、描述、分类、酬金、地点和截止时间，可选择上传任务图片，然后点击"发布"即可。

在任务列表中，找到状态为"待接取"的任务，点击"接取任务"按钮即可。每个任务只能被一个用户接取。

只有管理员可以删除任务。普通用户可以联系管理员处理。
